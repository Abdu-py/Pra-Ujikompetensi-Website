# Paket Tugas Praktik Demonstrasi — Marketplace Online

Isi paket ini:

```
paket-tugas/
├── mockup/
│   └── ujikom-app.drawio        # Rancangan UI (7 halaman: Login, Dashboard Admin,
│                                 # Kategori, Produk, Stok, Pesanan, Lihat Marketplace)
└── panduan-laravel/
    └── LANGKAH-LANGKAH.md       # Panduan step-by-step membangun aplikasi
                                  # Marketplace dengan Laravel 11 + MySQL
```

**Cara pakai:**
1. Buka `mockup/ujikom-app.drawio` dengan [draw.io / diagrams.net](https://app.diagrams.net) untuk melihat/menyunting rancangan UI.
2. Ikuti `panduan-laravel/LANGKAH-LANGKAH.md` untuk membangun aplikasi Laravel-nya dari nol.
